  1. 프로그램 다운로드

    • Releases · gohugoio/hugo · GitHub

  2. 압축 해제 후 Path 등록

  3. 블로그 폴더 생성

    hugo new site (폴더명)

  4. 블로그 폴더 이동 및 테마 파일 다운로드

    cd (폴더명)
git init
   
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
   
echo 'theme = "ananke"' >> config.toml

  5. 새 포스트 작성

    hugo new posts/my-first-post.md

  6. Hugo 서버

    hugo server -D
