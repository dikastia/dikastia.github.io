Micro SD 카드 만들기

다음 지시 사항에서 sdX 를 컴퓨터에 표시된 SD 카드의 장치 이름으로 바꾸십시오.

  1. SD 카드의 시작을 0으로 설정하십시오.:

    dd if=/dev/zero of=/dev/sdX bs=1M count=8

  2. fdisk를 시작하여 SD 카드를 분할하십시오.:

    fdisk /dev/sdX

  3. fdisk를 사용하여 새 파티션을 작성하십시오.:

    1. Type o. This will clear out any partitions on the drive.
    2. Type p to list partitions. There should be no partitions left.
    3. Type n, then p for primary, 1 for the first partition on the drive, 4096 for the first sector, and then press ENTER to accept the default last sector.
    4. Write the partition table and exit by typing w.

  4. Create the ext4 filesystem:

    mkfs.ext4 /dev/sdX1

  5. Mount the filesystem:

    mkdir root
mount /dev/sdX1 root

  6. Download and extract the root filesystem (as root, not via sudo):

    wget http://os.archlinuxarm.org/os/ArchLinuxARM-odroid-xu3-latest.tar.gz
bsdtar -xpf ArchLinuxARM-odroid-xu3-latest.tar.gz -C root

  7. Flash the bootloader files:

    cd root/boot
sh sd_fusing.sh /dev/sdX
cd ../..

  8. Unmount the partition:

    umount root

  9. Insert the micro SD card into the HC2, connect ethernet, and apply 12V power.

  10. Use the serial console or SSH to the IP address given to the board by your router.

    • Login as the default user alarm with the password alarm.
    • The default root password is root.

  11. Initialize the pacman keyring and populate the Arch Linux ARM package signing keys:

    pacman-key --init
pacman-key --populate archlinuxarm

Root HDD로 이동

이미 만들어진 Arch linux 활용

  1. 이미지를 HDD에 압축

    wget http://os.archlinuxarm.org/os/ArchLinuxARM-odroid-xu3-latest.tar.gz
bsdtar -xpf ArchLinuxARM-odroid-xu3-latest.tar.gz -C root

  2. Boot SDcard 생성

    dd if=/dev/zero of=/dev/sdX bs=1M count=8
fdisk /dev/sdX #최소 20MB이상 파티션 생성
mkfs.ext4 /dev/sdx1

  3. HDD의 boot\ 폴더를 sdcard로 복사

  4. boot\boot.txt 수정 (root=LABEL=root)

    setenv bootargs "console=tty1 console=ttySAC2,115200n8 **root=LABEL=rootfs** rw rootwait smsc95xx.macaddr=${macaddr} ${videoconfig}"

     pacman -S uboot-tools
 sh mkscr

  5. HDD 라벨 변경

    e2label /dev/sda1 rootfs
