Git GUI 한글깨짐 수정

  1. Gui 인코딩 utf-8로 변경

     git config --global gui.encoding utf-8

Git 자동 로그인 설정

  1. SSH key 생성

    ssh-keygen.exe -t rsa -C "이메일 주소"

  2. SSH 공개키 GitHub에 등록 (클립보드에 공개키 복사)

    cat ~/.ssh/id_rsa.pub | clip

  3. GitHub Personal Settings SSH keys 등록

  4. GitHub Remote 주소 변경 $ git remote -v

    View existing remotes

origin https://github.com/user/repo.git (fetch) origin https://github.com/user/repo.git (push)

$ git remote set-url origin https://github.com/user/repo2.git

Change the ‘origin’ remote’s URL
